# 🛒 Grozo

It is a  modern grocery delivery web application inspired by Blinkit, built using **React** and **Vite**. Grozo provides a fast, responsive, and user-friendly shopping experience with features like product search, category filtering, cart management, checkout, location selection, and order confirmation.

>  it is the  frontend-only grocery delivery application built for learning modern React development and UI/UX design.

---



##  Features

- 🏠 Modern Blinkit-inspired UI
- 🔍 Real-time product search
- 🗂️ Category-based filtering
- 🛍️ Add/Remove products from cart
- ➕ Quantity management
- 📍 Manual & current location selection
- 🔐 Login modal
- 🛒 Interactive cart drawer
- 💳 Checkout page
- 🎉 Order success confirmation
- 🔥 Toast notifications
- 💾 LocalStorage support
- 📱 Fully responsive design
- ⚡ Built with React + Vite

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Farhan8271git/grozo.git
```

Go to the project folder

```bash
cd grozo
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
